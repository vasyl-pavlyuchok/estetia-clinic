# Hero Media Request - Estetia (for Gem)

## Objective

Generate premium hero media for Estetia homepage with a laboratory-beauty aesthetic:
- mysterious but elegant,
- scientific but aspirational,
- visually rich without feeling aggressive.

## Required Outputs

1. Video loop (6-10s), 16:9, seamless.
2. Poster frame (16:9) from same scene.
3. Secondary still (4:5) for supporting card/composition.

## Scene Direction

- Macro/close shot of transparent serum in a pipette.
- Visible micro-bubble trapped in the liquid.
- Deep dark-lab background with controlled blue/cyan highlights.
- Professional cinematic lighting, premium skincare editorial tone.
- Camera mostly stable; subtle liquid movement only.

## Neuroesthetics Intent

- Peak Shift: slightly exaggerated glow/clarity in liquid texture.
- Isolation: one clear focal object (pipette + serum).
- Contrast: deep dark environment vs luminous fluid highlights.
- Generic Viewpoint: clean frontal/3-4 angle that feels honest, not gimmicky.

## Prompt Base (video/still)

`Photorealistic premium medical-beauty laboratory scene, macro close-up of a transparent serum inside a pipette with a trapped micro-bubble, dark elegant background, deep blue and cyan highlights, clean cinematic lighting, high-end skincare editorial look, subtle motion, ultra-detailed liquid texture, minimal composition, no text, no logos, no people, luxury aesthetics.`
