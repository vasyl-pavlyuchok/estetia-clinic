# Gem Collaboration Protocol

External collaborator: `Estetia - Web Design Expert` (Gemini).

## Why this collaborator matters

- Deep context of Estetia project direction.
- Can generate professional IA images aligned to brand goals.
- Can potentially convert image assets into professional video material.
- Has internet access for fresh references and trend validation.

## When to consult the Gem

Consult when one of these is true:

1. Visual direction is ambiguous between multiple premium approaches.
2. We need new hero/service imagery with strict aesthetic constraints.
3. We need short video loops or motion-friendly media concepts.
4. We need external inspiration grounded in current web/design landscape.
5. A copy/visual claim needs stronger creative framing before implementation.

## How to use efficiently

1. Define exact output needed (image set, visual style, framing, ratio, usage section).
2. Include target section and purpose (Hero, Services, AI showcase, etc.).
3. Request 2-3 variants max per batch to keep iteration tight.
4. Validate outputs against brief + neuroesthetics before implementation.

## Suggested ask template

`Need assets for <section>. Goal: <conversion/trust/desire outcome>.`
`Style constraints: <palette, lighting, tone, realism level>.`
`Please generate <N> variants + recommended usage notes (desktop/mobile).`
