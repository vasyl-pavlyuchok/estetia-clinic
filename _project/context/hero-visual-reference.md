# Hero Visual Reference (Estetia)

## Objetivo

Conservar y reutilizar el lenguaje visual aprobado en Hero para el resto de secciones y futuras páginas del sitio.

## Decisiones aprobadas

1. Dirección visual general:
- Estética premium, limpia, con contraste alto sobre media oscura.
- Evitar ruido visual innecesario.
- Priorizar legibilidad y jerarquía clara.

2. Video Hero:
- Fondo principal con video Cloudinary.
- Overlay oscuro + rejilla sutil para mantener legibilidad sin perder sofisticación.

3. Botones Hero (estilo serum estático):
- Botón izquierdo: serum dorado (más cálido/intenso).
- Botón derecho: serum azulado (más frío/intenso).
- Ambos con transparencia controlada y acabado premium (sin animaciones líquidas).

4. Texto superior (eyebrow):
- `Estetia Clinic` en blanco claro.
- Subtexto `Precisión Biomédica y Belleza Inteligente` con gradiente blanco + azul + dorado más visible.

5. Indicador de scroll:
- Solo forma interna tipo mouse (sin cápsula externa).
- Animación sutil de línea ascendente.

6. Efectos descartados:
- Burbujas sueltas decorativas en Hero.
- Animaciones “líquidas” complejas en botones del Hero.

## Regla de reutilización

Aplicar este lenguaje como base en nuevas secciones:
- mantener contraste y limpieza,
- usar acentos serum dorado/azulado de forma controlada,
- no saturar con efectos.

Si una nueva sección compromete legibilidad o claridad, reducir efecto antes de añadir más detalle visual.
